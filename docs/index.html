GH-900 教室

專為 Git 初學者設計的實作空間！

  • 🚀 GitHub Pages 手冊
  • 📝 GitHub Gists 手冊
  • 📚 GitHub Wiki 手冊
- README.md ： 儲存庫的門面，主要用來介紹專案目的、安裝流程與使用指南。 - GitHub Wiki ： 專案的百科全書，適合存放長篇的技術細節、開發規範或詳細的教學文件。 - GitHub Pages ： 專案的專屬網站，能將儲存庫中的靜態檔案直接轉化為對外公開的網頁。 - GitHub Gists ： 程式碼的便利貼，用於快速分享單一檔案、程式碼片段或輕量級的筆記。

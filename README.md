# GH900-Ch02
這是 README.md 可以顯示 較短 的訊息，單一檔



| 特性 | README.md | GitHub Wiki | GitHub Pages | GitHub Gists |
| :--- | :--- | :--- | :--- | :--- |
| **主要用途** | 專案簡介、快速上手 | 深度技術手冊 | 形象官網、Demo | 程式碼片段、筆記 |
| **內容長度** | 較短、精煉 | 長篇、多章節 | 完整網站架構 | 極短 (單一/少數檔案) |
| **技術門檻** | 極低 | 低 (Markdown) | 中 (需 HTML/CSS) | 極低 (直接貼上) |
| **自定義程度** | 低 | 中 (有側邊欄) | 極高 (完全客製) | 極低 (固定格式) |
| **私隱控制** | 隨專案設定 | 隨專案設定 | 全公開 | 可設為 Secret (隱藏) |

### 總結
* **README** 是書的封面。
* **Wiki** 是書的詳細章節。
* **Pages** 是專為這本書架設的專題官網。
* **Gist** 則是書中夾著的一張便利貼。

你目前在整理 Git 課程教材時，如果有某些**常用的設定檔片段**（例如特定的 `.gitignore` 模板），就很適合用 **Gist** 來存放並分享給學生！
